Phone URL settings!

URL to make call to

This is where you can input your url that you wish to connect the call through you can use the {phone} switch in your url to input the clicked phone number for example the phone number clicked is 719-550-2230. And your url is http://TestPhone.com/Call.php?to={phone}&from=112. When you click the call button it will call the url http://TestPhone.com/Call.php?to=7195502230&from=112.
